# Task Tracker

A simple task tracker application to create, update, and manage tasks.

## Features

- Add, edit, and delete tasks
- Mark tasks as complete/incomplete
- Filter and search tasks
- Lightweight and easy to extend

## Getting Started

These instructions will get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

- Git
- Node.js and npm (if the project uses a Node-based toolchain)

### Installation

1. Clone the repository:
   git clone https://github.com/aanandchhimpa/Task-Tracker.git
2. Change into the project directory:
   cd Task-Tracker
3. Install dependencies (if applicable):
   npm install
4. Start the app (if applicable):
   npm start

Note: If this repository is a static front-end project, you can open index.html in your browser instead.
